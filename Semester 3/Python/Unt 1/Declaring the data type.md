### Numeric Data Types (int, float, complex):
```python
# Integer
my_int = 5

# Float
my_float = 3.14

# Complex
my_complex = 2 + 3j
```

### String Data Type:
```python
# String
my_string = "Hello, World!"
```

### Boolean Data Type:
```python
# Boolean
my_bool = True  # Can be True or False
```

### List Data Type:
```python
# List
my_list = [1, 2, 3, 4, 5]
```

### Tuple Data Type:
```python
# Tuple
my_tuple = (10, 20, 30)
```

### Set Data Type:
```python
# Set
my_set = {1, 2, 3, 4}
```

### Dictionary Data Type:
```python
# Dictionary
my_dict = {'name': 'Alice', 'age': 30}
```
